# Main layout components

Компоненты страницы `/main` (после авторизации).

- **Sidebar** — аватар (круг, инициалы при отсутствии), имя/фамилия, при наведении на аватар — иконка карандаша, клик — загрузка фото; снизу кнопка «Выйти».
- **ChatList** — список чатов: карточка = круг (аватар чата) + название, снизу последнее сообщение (пока заглушка); клик — выбор чата.
- **Chat** — выбранный чат: сообщения (свои справа, чужие слева), имя отправителя для групповых; снизу `react-textarea-autosize` и отправка. Время с полной датой в `title` при наведении.
